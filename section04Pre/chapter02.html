React Application?

React로 만든 웹 서비스들은 보통 React App, React Application 이라고 불리움
** 그 이유는 단순한 웹을 뛰어넘어 어플리케이션에 가까운 기능을 선보이기 때문
React 또한 Node.js의 NPM 라이브러리 중 하나이기 때문에 아래와 같은 순서로 React App을 생성할 것이다.
1. Node.js 패키지 생성
2. React 라이브러리 설치
3. 기타 도구 설치 및 설정 (입문자에게는 권장하기 어려움)
** 설정의 어려움을 극복하기 위해 Vite라는 차세대 프론트엔드 개발 툴을 통해 기본적인 리엑트 설정이 미리 완료된 상태의 React 앱을 생성할 것이다.

아래는 vite를 통하여 프로젝트를 생성하는 예제이다.

jaemyung@choejaemyeong-ui-MacBookAir onebite-react % npm create vite@latest
